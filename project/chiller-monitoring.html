Project 2 - Chiller Monitoring

Chiller Monitoring merupakan projek untuk memonitor suhu cairan chiller pada PT Toshin Prima Fine-Blanking, sebagai projek magang. Monitoring ini untuk tindakan preventif jika ada kenaikan suhu pada salah satu chiller. Chiller Monitoring terdiri dari alat monitoring suhu, alarm, dan UI monitoring Suhu

Berikut merupakan desain casing dari Alarm dan Monitoring suhu. Dan juga alat monitoring suhu. Terdiri dari :

  • ESP32 untuk mengirimkan nilai suhu pada rangkaian alarm dan aplikasi monitor untuk ditampilkan
  • Relay untuk menghidupkan rangkaian bila tersambung dengan internet

Berikut merupakan tampilan monitoring dari suhu chiller. Menggunakan teknologi node red yang terhubungan dengan internet, data dari sensor dikirim melalui internet via Firebase lalu ditampilkan, nilai suhu juga berpengaruh pada warna gauge
