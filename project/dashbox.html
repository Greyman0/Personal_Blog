Project 1 - Dashbox

Dashbox merupakan alat yang digunakan untuk mendukung simulasi, simulasi di sini ialah berbentuk permainan simulasi.Alat ini terhubung dengan aplikasi SimHub untuk menghubungkan dengan permainan simulasi pada PC. Gunanya untuk meningkatkan feel dalam bermain.

Dashbox terdiri dari alat dan fungsinya masing-masing, yaitu :

  • Push Button : Button dapat diatur untuk memiliki fungsi tertentu pada simulasi, dapat diatur dengan SimHub
  • Rotary Encode : dapat digunakan untuk mengurangi atau menambahkan sesuatu dalam simulasi secara teratur, contohnya mengatur Braking Bias pada Game F1 2020
  • Display : memberika visualisasi pada simulasi seperti Gear, kecepatan, rev light

Berikut adalah Schematic Dashbox ,terdiri dari:

  • Arduino Nano
  • Push Button
  • Encoder EC11
  • TM1637
  • WS2812 8 bit
  • MAX1637
